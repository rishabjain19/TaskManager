# Task Manager

A simple and responsive Task Manager application built using React.  
This project helps users create, manage, and organize their daily tasks efficiently.

## Features

- Add new tasks
- Delete existing tasks
- Reorder tasks (move up and move down)
- Clean and minimal user interface
- Fast and responsive performance

## Tech Stack

- React
- JavaScript (ES6+)
- HTML5
- CSS3
- Vite (for project setup and bundling)
